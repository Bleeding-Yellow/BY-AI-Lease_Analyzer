# Bleeding-Yellow AI Lease Analyzer

**A fully client-side web application** that extracts key lease information from PDF, Word, TXT, and Excel files using pure JavaScript (pdf.js + mammoth.js + SheetJS + regex AI parser).

No backend, no installation, no API keys. Just open the HTML file in your browser.

## Features
- Upload up to **3 files at once** (PDF, .doc/.docx, .txt, .xls/.xlsx)
- Add files with the big "+" button or drag & drop
- Extracts: Lease ID/No, Lease Name, Start Date, End Date, Status, File creation date (browser metadata), Signer(s)
- Full extracted text viewer
- Beautiful professional UI with Bleeding Yellow branding
- 100% offline — perfect for GitHub demo / portfolio

## How to Run (Super Simple)
1. Download **BY-AI-LA-index.html**
2. Double-click the file (or right-click → Open with Chrome/Firefox/Edge)
3. Upload your lease documents and click **"ANALYZE WITH AI"**

**Test it immediately** with the included sample file.
